| Persona | 대상 | Pain Point | 핵심 니즈 | 전환 트리거 |

## 2. User Journey & Conversion Funnel

### 2.1 Principal/Teacher Journey (B2B)
```
[인지] Google 검색 "Korea school trip India"
  → Landing Page (Why Korea? or For Schools)
    → [관심] Program Proposal PDF 다운로드 (리드 수집)
      → [비교] Tour Packages 상세 비교
        → [검증] FAQ + Safety 페이지 확인
          → [행동] Schedule a Meeting / Request Quote
            → [전환] 미팅 후 계약
              → [추천] 참여 후기 작성 → 다른 학교 소개
```

### 2.2 Parent Journey
```
[인지] 학교로부터 프로그램 안내 수신 또는 SNS 광고
  → Landing Page
    → [관심] FAQ (안전, 음식, 비용) 확인
      → [비교] 패키지별 가격/포함 사항 비교
        → [검증] Gallery + Testimonials 확인
          → [행동] WhatsApp 문의 또는 학교에 참여 의사 전달
```

### 2.3 Student Journey
```
[인지] Instagram Reel / YouTube 영상 / 친구 추천
  → Blog 콘텐츠 ("K-pop dance in Korea" 등)
    → [관심] Experiences 페이지 탐색
      → [공유] 부모에게 링크 전달
        → [행동] 학교에 프로그램 요청
```

### 2.4 핵심 전환 포인트 (Conversion Points)
| 전환 지점 | 위치 | 목표 전환율 |
|-----------|------|-----------|
| WhatsApp 문의 | 모든 페이지 플로팅 버튼 | 3-5% |
| Proposal PDF 다운로드 | For Schools 페이지 | 15-20% |
| Request Quote 폼 제출 | Tour Packages 페이지 | 5-8% |
| Meeting 예약 | For Schools 페이지 | 2-3% |
| Newsletter 구독 | Blog / Footer | 5-10% |

### 4.2 Why Korea?

#### 섹션 1: India-Korea Connection (신규)
- 한국-인도 수교 관계 & 경제 협력 확대
- 인도 학생의 한국 유학 증가 추세
- 한류(K-Wave)의 인도 내 인기 폭발
- 양국 IT/기술 협력 (삼성 인도 공장, 현대 인도 시장 등)
- "Korea is not a foreign country — it's India's growing partner"

#### 섹션 2: 6 Pillars of Why Korea

**1. Global Education Powerhouse**
- PISA 순위 상위권, 세계적 대학 랭킹
- 캠퍼스 방문: Yonsei, SNU, KAIST, KMOU, Busan National, Ewha
- 학생들이 직접 커리큘럼 체험 & 유학 상담 가능
- *인도 학생에게 의미: "한국 대학 유학 경로를 직접 탐색"*

**2. Technology & Innovation Leader**
- 삼성, 현대, LG, SK 등 글로벌 기업 본사
- 세계 최고 수준의 인터넷/5G 인프라
- 방문: KARI(항공우주), KAIST, 로봇공학센터, 삼성뮤지엄, 현대공장
- *인도 학생에게 의미: "STEM 진로에 대한 구체적 비전 획득"*

**3. 5,000 Years of Living History**
- 궁궐, 한옥마을, 전통 공예 거리가 현대 도시 속에 살아있음
- 김치 만들기(유네스코 무형유산), 한복 체험, 태권도
- *인도 학생에게 의미: "전통과 현대의 공존 — 인도와 유사한 과제"*

**4. Real-World Geopolitics Education (DMZ)**
- 세계 유일의 분단 현장, 냉전의 마지막 전선
- 임진각, 제3땅굴, 도라전망대 방문
- 전쟁 역사, 평화 연구, 국제관계를 현장에서 학습
- *인도 학생에게 의미: "교과서 밖의 국제정치를 체험"*
- **이것은 싱가포르/태국에서 절대 불가능한 경험**

**5. K-Culture Global Influence**
- K-pop 댄스 체험 + 뮤직비디오 촬영
- K-beauty, K-food 체험
- 난타/페인터즈 히어로 공연 관람
- KBS 드라마 스튜디오 방문
- *인도 학생에게 의미: "좋아하는 K-culture를 본고장에서 체험"*

**6. Safe & Student-Friendly**
- 세계 안전 지수 상위 (Global Peace Index)
- 범죄율 매우 낮음, CCTV 촘촘
- 우수한 대중교통, 의료 시스템
- WiFi/모바일 연결성 최상 → 학부모와 실시간 소통
- *인도 학부모에게 의미: "가장 안전한 곳에서 가장 많이 배우는 여행"*

### 4.10 AI Content Automation & Naver Blog Integration (핵심 신규 기능)

#### 4.10.1 기능 개요

**목적:** AI가 한국 여행 관련 콘텐츠를 자동으로 생성하고, 네이버 블로그에 발행하면서 동시에 메인 홈페이지에 실시간 노출하여 지속적인 방문자 유입과 관심 유도

**왜 이 기능이 핵심인가:**
- 여행사 홈페이지는 패키지 정보만으로는 **재방문율이 낮음**
- AI가 매일/매주 새로운 콘텐츠를 생성 → **홈페이지가 항상 살아있는 느낌**
- 네이버 블로그는 **한국 내 검색 점유율 1위** → 한국 측 SEO 확보
- 메인 페이지에 실시간 피드 → 방문자가 **"이 회사는 활발하게 운영중"** 이라는 신뢰감
- 시즌별/트렌드별 콘텐츠 자동 대응 → **인력 없이 콘텐츠 마케팅 운영**

#### 4.10.8 AI 콘텐츠 품질 관리

**A. 톤앤매너 가이드라인 (AI System Prompt에 포함)**
```
- Target Reader: Indian high school students, parents, teachers
- Tone: Friendly, informative, enthusiastic but professional
- Always include: Practical tips, safety mentions, Indian food options
- Never include: Political opinions, religious comparisons, negative stereotypes
- Cultural sensitivity: Respect both Indian and Korean cultures
- CTA: Every post ends with a related package or "Contact us" link
- SEO: Natural keyword insertion, no keyword stuffing
- Length: Concise, scannable, use bullet points and headers
```

**B. 자동 품질 체크 파이프라인**
```
Content Generated
  → 금칙어 필터 (정치/종교/민감 표현)
  → 팩트체크 (가격, 날짜, 장소 정보 DB 대조)
  → SEO 스코어 체크 (키워드 밀도, 메타 길이)
  → 중복 콘텐츠 검사 (기존 발행 글과 유사도)
  → 통과 → 자동 발행 또는 관리자 큐에 추가
  → 불통과 → 재생성 또는 수동 검토 알림
```

**C. 콘텐츠 성과 피드백 루프**
```
발행 후 7일 → 성과 분석
  → 조회수, 체류시간, CTA 클릭률 측정
  → 고성과 주제/키워드 → AI에 피드백 (다음 생성에 반영)
  → 저성과 주제 → 자동 감소 또는 각도 변경
  → 월간 성과 리포트 자동 생성
```

### 5.4 UI/UX Principles
- **Mobile First**: 인도 웹 트래픽 75%+ 모바일
- **Thumb-Friendly**: 모바일에서 터치 영역 48px 이상
- **Fast Loading**: 인도 3G/4G 네트워크 고려
  - 이미지: WebP, AVIF 포맷 + srcset 반응형
  - Lazy loading 모든 Below-the-fold 이미지
  - Critical CSS inline
  - 초기 로드 < 200KB
- **WhatsApp Integration**: 모든 페이지 플로팅 버튼 + 페이지 내 CTA
- **Trust Signals**: 대학 로고, 학교 참여 수, 안전 인증, 실사진
- **Progressive Disclosure**: 처음엔 핵심만 → 클릭으로 상세 확장
- **Sticky Navigation**: 스크롤 시 축소되는 네비게이션
- **Breadcrumb**: 패키지/체험 상세 페이지에서 위치 표시

### 5.5 Key Visual Elements
- 한국 전통 패턴 (단청, 기와) 을 현대적으로 재해석한 섹션 구분선
- 도시별 커스텀 아이콘 (서울: N타워, 부산: 광안대교, 대전: KAIST, 제주: 한라산)
- 인도 학생들이 한국에서 활동하는 실제 사진 (**핵심 에셋 - 촬영 필수**)
- Day Timeline 인포그래픽 (세로 타임라인 + 아이콘)
- Before/After 슬라이더 (전통 궁궐 ↔ 스마트 시티)
- 인터랙티브 한국 지도 (도시 클릭 → 해당 체험 표시)
